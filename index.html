Pinjar - Pembelajaran Interaktif AR

Deskripsi Project

Pinjar adalah aplikasi pembelajaran berbasis Augmented Reality yang dirancang untuk meningkatkan pengalaman belajar siswa melalui visualisasi 3D interaktif. Aplikasi ini dikembangkan untuk memudahkan proses pembelajaran dengan teknologi AR.

Teknologi yang Digunakan

  • Unity 3D Engine
  • Vuforia AR SDK
  • C# Programming
  • Android SDK
  • Blender untuk Modeling 3D

Fitur Utama

  • Eksplorasi objek 3D tanpa alat peraga fisik
  • Visualisasi interaktif untuk pembelajaran
  • Integrasi dengan smartphone untuk kemudahan akses
  • Interface yang user-friendly untuk guru dan siswa
  • Sistem tracking objek AR yang akurat

Hasil & Pencapaian

  • Tingkat kepuasan pengguna mencapai 92% berdasarkan uji kuisioner
  • Berhasil mengadaptasi teknologi AR untuk pembelajaran
  • Meningkatkan minat belajar siswa melalui metode pembelajaran baru
  • Memudahkan eksplorasi objek pembelajaran tanpa keterbatasan alat peraga
  • Dinyatakan layak sebagai media pembelajaran baru bagi siswa

Metodologi Pengembangan

  • Analisa Kebutuhan
  • Perancangan Sistem
  • Development
  • Testing & Implementasi
  • Evaluasi menggunakan metode skala likert

Kluwarga - Portal Komunitas Desa

Deskripsi Project

Kluwarga adalah platform digital yang dirancang khusus untuk memudahkan interaksi dan pengelolaan informasi dalam komunitas desa. Website ini menjembatani komunikasi antara perangkat desa dengan warga, serta memfasilitasi berbagai kegiatan administratif desa secara digital.

Teknologi yang Digunakan

  • PHP dengan Framework CodeIgniter
  • MySQL Database
  • Bootstrap untuk UI
  • JavaScript & jQuery
  • REST API Integration

Fitur Utama

  • Manajemen Berita & Pengumuman Desa
  • Forum Diskusi Warga
  • Sistem Pengaduan Online
  • Kalender Kegiatan Desa
  • Administrasi Surat Menyurat Digital
  • Dashboard Admin untuk Pengelolaan Konten

Hasil & Dampak

  • Peningkatan efisiensi komunikasi antar warga
  • Digitalisasi proses administratif desa
  • Transparansi informasi kegiatan desa
  • Kemudahan akses layanan desa bagi warga
